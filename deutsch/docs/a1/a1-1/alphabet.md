### Vocales con Umlaut (Diéresis) y Eszett

| Letra | Pronunciación Aproximada* |
| :---: | :--- |
| **Ä** | *a-e* |
| **Ö** | *o-e* |
| **Ü** | *o-u* |
| **ß** | *ss* |

!!! tip "Truco de pronunciación para los Umlauts (*)"
    Para las letras marcadas con asterisco (Ä, Ö, Ü), la imagen sugiere esta técnica:
    
    > **La 1ª vocal indica la forma de la boca, la 2ª vocal es el sonido que intentas hacer.**
    
    *Por ejemplo para la **Ö**: Pones boca de **O**, pero intentas decir **E**.*
